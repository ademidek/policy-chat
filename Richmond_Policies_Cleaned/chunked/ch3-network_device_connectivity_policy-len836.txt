the university provides an anonymous and unencrypted wi-fi network for visitors. there may be some

classification of networked devices
workstations
workstations are general use devices that run microsoft windows, macos, or linux operating systems.
servers
servers are devices that provide shared resources such as file shares, printers, or an application to one or
more users over the network
